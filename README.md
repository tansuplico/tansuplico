## What I'm doing right now

I'm taking **APIIS LMS** — a Learning Management System built for rural Philippine communities — from a Tauri desktop app to a Railway-hosted web app, and getting it ready for real-world deployment.

Most of my recent work has been the unglamorous-but-essential kind: hardening offline behavior so the app doesn't quietly break when a connection drops, auditing the backend for security gaps, and tearing out features that sounded good on paper but turned into dead weight in practice.

**Stack:** React · TypeScript · TailwindCSS · Zustand · Node.js · Express · PostgreSQL · SQLite

A few things I actually had to solve:

- 🔌 **Offline-first by design, not as an afterthought** — students can keep learning and have their progress sync automatically once they're back online
- 🔐 **Security pass across every controller** — closed ownership-check gaps, race conditions, and a path traversal issue before they became real problems
- 🎥 **Token-scoped video streaming** — short-lived JWTs instead of broken auth on every stream request
- 🧹 **Said no to scope creep** — scrapped a half-built video download feature and a per-user thumbnail system once it became clear they weren't worth the complexity they added
